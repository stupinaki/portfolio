## Приветствую тебя, путник!

Это сайт-визитка, здесь ты найдешь ссылки на мои проекты с описанием, а так же примеры кода.

## Code style

1. Названия компонентов: `PascalCase`
2. Стили: `module.scss`
3. Название файлов со стилями: `camelCase`
4. ClassName: `camelCase`
5. Если svg импортируется как React компонент, то название в `PascalCase`, иначе в `camelCase`

## Secrets

1. .env.local — всё что секретно убираем сюда, не должно быть в репозиторий
2. .env — всё что не секретно, храним здесь
3. .env.example — хранит перечень ключей, которые нужно будет запросить у коллег
